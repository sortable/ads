# Use Sortable Plugin

### Add CDN Link for Sortable Service Script

```html
<html>
  <head>
    <script async src="//tags-cdn.deployads.com/a/<SITE_NAME>.js"></script>
  </head>
</html>
```

### Enable Sortable Plugin

```js
sortableads.push(function() {
  // configuration ...
  sortableads.useSortableForGPTAsync();

  // ...
});
```
